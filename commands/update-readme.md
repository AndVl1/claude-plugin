# Update README

Update the README.md file to reflect the current state of the project after significant changes.

## When to Use

Run this command after:
- Adding new features or bot commands
- Changing project structure
- Updating dependencies
- Modifying environment variables
- Adding new configuration options

## Instructions

1. **Read current state**:
   - Check README.md for existing content
   - Review recent changes in the codebase
   - Look at new files and modified modules

2. **Update sections**:
   - **Features**: Add/remove capabilities
   - **Commands**: Update bot command table
   - **Structure**: Reflect new packages/files
   - **Quick Start**: Update setup instructions if changed
   - **Environment Variables**: Add new required vars

3. **Keep it concise**:
   - README should be brief and practical
   - Link to CLAUDE.md for detailed technical info
   - Avoid duplicating information

4. **Verify accuracy**:
   - Check that all listed commands actually exist
   - Verify environment variable names match application.yml
   - Ensure project structure matches reality
